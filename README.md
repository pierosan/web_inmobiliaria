# Pagina web de agente inmobiliario

1. Para crear un entorno virtual
python -m venv venv

2. Entrar en el venv
cmd: venv\Scripts\activate.bat
powershell: venv\Scripts\Activate.ps1
bash: source venv/bin/activate

4. instalar libreria
pip install django

5. Para ejecutar el servidor
python manage.py runserver

link del sitio administrativo:
http://127.0.0.1:8000/admin

usuario: admin 
pass: admin
